Java has 8 primitive data types: byte (8-bit), short (16-bit), int (32-bit), long (64-bit).

float (32-bit) and double (64-bit) are floating-point types.

char is 16-bit Unicode character.

boolean has only true or false values.

Default values: int=0, boolean=false, object references=null.

Variables must be declared with a type before use.

Local variables must be initialized before use.

Final keyword makes a variable constant (cannot be reassigned).

Type casting: implicit (widening) and explicit (narrowing).

String concatenation uses + operator.
