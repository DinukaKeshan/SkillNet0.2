React applications are built using components.

Components are reusable, independent pieces of UI.

There are two main types: functional components and class components (functional are preferred since React 16.8+).

A functional component is a JavaScript function that returns JSX.

Components can receive data via props (properties).

Props are read-only and passed from parent to child.

The component name must start with an uppercase letter.

Components can be nested to build complex UIs.

The root component is usually rendered in index.js using ReactDOM.

Components can have their own state using useState hook in functional components.
