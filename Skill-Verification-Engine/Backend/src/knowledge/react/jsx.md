JSX stands for JavaScript XML.

JSX allows writing HTML-like code inside JavaScript.

It is syntactic sugar that gets transpiled to React.createElement() calls.

JSX expressions are written inside curly braces {}.

Attributes in JSX use camelCase (e.g. className instead of class).

Self-closing tags are required for elements without children (e.g. <img />).

JSX prevents injection attacks by escaping values by default.

You can embed JavaScript expressions, variables, and functions in JSX using {}.

Conditional rendering can be done with &&, ternary operator, or if statements outside JSX.

Lists in JSX require a unique key prop on each element for efficient reconciliation.
