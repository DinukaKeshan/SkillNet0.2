Java is a high-level, object-oriented, platform-independent programming language.

Java code is compiled to bytecode and runs on JVM (Java Virtual Machine).

Write once, run anywhere (WORA) is achieved via bytecode and JVM.

Main method entry point: public static void main(String[] args).

Java is strongly typed and case-sensitive.

Primitive data types: byte, short, int, long, float, double, char, boolean.

Wrapper classes: Integer, Double, Boolean, etc. for primitives.

String is immutable and part of java.lang package.

System.out.println() is used for console output.

Java uses single-line // and multi-line /* */ comments.
